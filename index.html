History

Andrea Cimitan wrote zencoder in C#, a video encoding software for the creative zen. I like the simplicity of zencoder, therefore I took the source and modified it, so I can use it for my Android phone.

Download

Please be sure, you have a working copy of ffmpeg with libx254 and mpeg4 support. Under Ubuntu 9.04 (Jaunty Jackalope) you can do this by running: 

$ sudo apt-get install libavcodec-unstripped-52 \
libavdevice-unstripped-52 libavformat-unstripped-52 libavutil-unstripped-49 \
libpostproc-unstripped-51 libswscale-unstripped-0 ffmpeg

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/cit/Encodroyd

Installation

You can compile it by yourself by running: 

$ make clean; make
$ mono encodroyd.exe

get the source code on GitHub : cit/Encodroyd
